# AgentServer 应用说明

**更新日期：** 2026-04-10

## 当前定位

`agent_server` 是面向玩家/agent 运行时流程的原型层。它已有部分流程，但整体仍偏 demo/placeholder。

## 当前已实现

- 通过 `beacon_server` 加入集群
- 等待 `agent_manager`
- 启动 agent 进程并提供简化角色选择流程

## 当前限制

- 核心 agent 数据路径仍有硬编码 / demo 色彩
- 角色列表和状态还没有一个稳定的持久化真源

## 开发提示

- 在扩展功能前，应先明确角色选择和 agent state 的最终数据来源
- 可结合根项目迁移文档理解其在整体架构中的位置
