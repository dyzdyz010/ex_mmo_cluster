# AgentManager 应用说明

**更新日期：** 2026-04-10

## 当前定位

`agent_manager` 是 `agent_server` 的集群可见协调外壳，目前仍处于非常轻量的阶段。

## 当前已实现

- 通过 `beacon_server` 加入集群发现
- 注册 `:agent_manager` 资源
- 为后续 agent 协调逻辑提供稳定挂点

## 当前限制

- 还没有真正落地 placement、pooling、调度、生命周期协调等管理能力
