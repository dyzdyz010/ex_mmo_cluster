# DataInit 应用说明

**更新日期：** 2026-04-10

## 当前定位

`data_init` 是旧 Mnesia schema / bootstrap 工具库，属于迁移期基础设施。

## 当前已实现

- Mnesia schema 创建辅助
- 表创建 / 复制辅助
- 旧数据拓扑依赖的 database copy 逻辑

## 当前职责

在仓库尚未完全切换到 PostgreSQL-first 之前，它仍然支撑部分旧拓扑启动流程。
