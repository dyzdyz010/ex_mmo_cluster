# coordinate_system NIF 说明

**更新日期：** 2026-04-10

## 项目定位

该子项目是 `scene_server` 的 Rustler NIF crate，对应模块 `Elixir.SceneServer.Native.CoordinateSystem`。

## 构建方式

- NIF 会随 `scene_server` 一起构建
- 常规情况下直接在 umbrella 根目录执行 `mix compile` 即可

## Elixir 侧加载方式

```elixir
defmodule Elixir.SceneServer.Native.CoordinateSystem do
  use Rustler, otp_app: :scene_server, crate: "coordinate_system"
end
```

## 说明

- 该 crate 主要服务于场景侧性能敏感逻辑
- 具体职责应与 `scene_server` 运行时文档配套阅读
