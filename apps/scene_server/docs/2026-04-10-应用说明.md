# SceneServer 应用说明

**更新日期：** 2026-04-10

## 当前定位

`scene_server` 是当前游戏运行时里最核心的场景侧服务之一，负责玩家角色、AOI、物理与运动循环。

## 当前已实现

- 玩家角色管理
- movement 同步路径
- AOI 管理
- 物理 / NIF 相关场景操作
