# VisualizeServer 应用说明

**更新日期：** 2026-04-10

## 当前定位

`visualize_server` 是 Phoenix/LiveView 观察层，目前更偏向开发者/运营侧场景查看工具，而不是完整产品界面。

## 当前已实现

- Phoenix endpoint
- `/scene` LiveView 路由
- 对 `SceneServer.PlayerManager` 的周期轮询与可视化展示

## 当前限制

- 当前更像调试/观察壳层，功能仍然较浅

## 本地开发

```bash
cd apps/visualize_server
mix phx.server
```
