# AuthServer 应用说明

**更新日期：** 2026-04-10

## 当前定位

`auth_server` 是基于 Phoenix 的认证入口，既承担浏览器登录入口，也承担面向 gate 的 token 校验能力。

## 当前已实现

- `/ingame/login` 登录流入口
- `AuthServer.Interface` 的集群注册与服务可见性
- `AuthServer.AuthWorker` 的 token 签发 / 校验
- 可被 `gate_server` 远程调用的校验表面

## 当前限制

- 账户/业务校验逻辑仍未完全成型
- 一部分账户读取仍依赖旧 `DataInit.TableDef` 结构
- 当前更像是“认证链路打通”，不是完整生产级账户系统
