> 本稿对该口径做**终态化修订**:投影路线即终态,同构路线降格为特定负载画像下的**定向优化选项**,不再是"终态"候选。

## 1. 两条路线的精确定义(用 glossary 术语)

- **投影路线**:truth 只在服务端;客户端是 snapshot-only 消费者——近窗拿 1m 投影(`0x62 ChunkSnapshot` + `0x63` delta 流),远区拿 7m 投影(source pages);客户端对世界生成算法零知识,配方(`base ⊕ overlay`)不跨 wire。
- **同构路线**:客户端持有与服务端 bit-exact 的 base 算法,服务端只下发 overlay(压扁修改层),客户端本地重建投影;要求跨 Rust NIF / Voxia C++ 浮点级一致并永续同步演化。

## 2. 裁决

1. **投影路线为终态路线**,覆盖近窗与远区两段——不是"当前阶段的权宜",不是等待同构条件成熟的过渡态。近窗继续全量投影下发(窗口内每体素在客户端持有合并态,这是任何方案的共同终态,见 glossary 推论 4);远区走 source pages(7m mip)。
2. **同构路线降格为定向优化选项**:仅当 §4 负载画像全部命中且评审稿五条件全绿时,以"**处女地基底本地生成**"的**定向加法**形态引入(被修改区仍走投影,gate/渲染管线不变),永不作为整体路线切换。
3. **客户端 WorldGen(C++ 副本)永久定位**:`-VoxiaWorldGenPreview` 开发预览 + 里程碑 A/B 的 fixture 数据源。它与服务端的 3D 分歧(lowland 振幅、mask 区间、独有浮空岛)**不再是需要偿还的债务**——"接连接态前重建跨端一致"的旧口径作废;连接态下客户端 WorldGen 不上场,自由实验空间永久保留。
4. H gate 基础设施保留,其校验职能延续现状定位:从"WorldGen fixture + D 签名"(同构时代设计)迁移到"pages manifest + sha256 + diff chain"(投影时代职能)。6-30 计划中"客户端本地推导 confirmed baseline"目标由本稿**正式关闭**(该文档按仓库纪律不回改,以本稿为准)。

## 3. 裁决依据(五维度)

1. **同构的带宽优势吃三重折扣**:① 本游戏立身之本是修改(编辑+涌现+设计师 3D 内容归 delta,W-Q6=A)——overlay 从 day one 厚且单调变厚,修改率越高同构优势越向投影收敛;② 远区已被 A-4 论证死(mip 需合并后状态),同构的节省只在近窗兑现,而近窗未修改 chunk 被压缩+范围声明打薄后仅 MB 级;③ 带宽是贬值资产,跨端 bit-exact 纪律(消 `powf`、FMA 锁定、fixture CI、版本协商)是每次内容迭代重付、随平台矩阵扩张的升值负债。
2. **内容演化自由**:投影路线把"世界是怎么来的"变成服务端私有实现细节——热更生成算法、离线管线、AI 创世、authored 大陆(元决策方向),客户端全部无感、数据滚 revision 即可。同构路线把世界内容编译进客户端二进制,内容迭代被发版节奏绑架。行业旁证:同构路线成功案例集中于"世界不可修改或弱多人"象限(No Man's Sky 型);可修改+强多人象限的工业收敛解即服务端权威+投影下发。
3. **信息不对称护城河**:同构=客户端持有完整生成算法=玩家可离线穷举全世界矿脉/地下结构/稀有资源(Minecraft seed 工具生态为现实先例),对资源经济型 MMO 是设计层漏洞而非实现层作弊。投影路线下世界的秘密保持服务端私有,勘探玩法、地理经济、信息差交易的上限只在投影下存在。
4. **一致性面:零 vs 乘法扩张**:投影的一致性面恒为零(数据过 sha256 即正确);同构的一致性负债=算法版本 × 平台 × 编译器 × CPU 特性(FMA),近窗 1 bit 分歧即"玩家站在服务端不存在的地面上"(碰撞直查体素)。本仓已用行动投票:客户端 WorldGen 为视觉迭代故意分叉——同构路线等于宣布该迭代回路永久关闭。
5. **不可逆性不对称**:投影→同构是加法(投影消费管线不动,为处女地加一条本地生成源;H gate 地基还在);同构→投影是重写(客户端数据源层重建,fixture CI/版本协商沉没)。走错的代价不对称,独立支持先走投影。

**天花板类型总结**:投影路线的瓶颈是服务端算力与分发成本——可横向扩展、可用钱买、CDN 可缓存(bounded materialization 预算门禁已在管);同构路线的瓶颈是跨端一致性——不能用钱买,只能用纪律永续维持。能用机器解决的瓶颈优于要用纪律扛住的瓶颈。

## 4. 同构路线的适用画像(写死;**全部**命中才重开评估)

1. 玩法形态转向:大陆尺度荒野为主、远景可见修改率稀疏、高速移动(飞行游猎型)为主——即"世界不变"重新成为常态;
2. 实测证据:近窗流送带宽在 observe 中成为可测瓶颈(评审稿五条件之⑤,`page_refetch_bandwidth` 等字段由数据回答);
3. 评审稿五条件①-④全绿(golden fixture CI 常绿一个发布周期、3D 内容归属重拍或基底信息占比测算、mip 派生规则跨端 fixture、运行时 kill-switch 可整体切回投影);
4. 引入形态限定:处女地基底本地生成的定向加法,被修改区仍走投影,失败语义不变(缺源硬失败,本地生成结果不得充当缺页 fallback)。
