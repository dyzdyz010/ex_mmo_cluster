## 摘要

在本仓体素权威化主线 §1 目标三（`docs/2026-05-07-体素服务器权威化架构进度检查.md` 的"为体素格填充属性"）的 typed attribute domain 落地之后，引入一层瞬时、稀疏、tick 驱动的局部场层（`FieldLayer`），作为现有 `MicroLayer` 的兄弟概念。以电场（导电/电离）与温度场（扩散）两类场作为骨架样例。

## 排序

本目标的前置依赖是体素权威化主线 §1 目标三：

- `AttributeSet` / `TagSet` typed domain 与 catalog/patch 协议（缺口 A）
- `temperature` / `moisture` / `density` / `thermal_conductivity` 四个 typed attribute 入 `ChunkSnapshot` / `ChunkDelta`
- `chunk hash` 覆盖 attribute / tag / environment
- `DataService` 持久化覆盖同一份 canonical snapshot

主线完成前，本目标只在 spec / bevy 原型阶段，不进入服务器实现。
